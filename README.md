### 🚀 Featured Projects

---

**UniWallet — Mobile Money Aggregator**

*NestJS · gRPC · Provider Abstraction Layer*

A Ghanaian mobile money aggregator unifying MTN MoMo, Telecel Cash, and AirtelTigo behind a single API, built around a Provider Abstraction Layer with gRPC for provider communication.

![Fintech](https://img.shields.io/badge/-Fintech-333333?style=flat-square) ![Backend](https://img.shields.io/badge/-Backend-333333?style=flat-square) ![gRPC](https://img.shields.io/badge/-gRPC-333333?style=flat-square) ![Mobile Money](https://img.shields.io/badge/-Mobile%20Money-333333?style=flat-square)

<br/>

**VoteFlow — Voting SaaS for West Africa**

*Next.js · Socket.io · Paystack · Arkesel*

A general-purpose voting platform for West African organizations and events, with real-time results via Socket.io, Paystack billing, and Arkesel-powered SMS/OTP verification.

![SaaS](https://img.shields.io/badge/-SaaS-333333?style=flat-square) ![Real--time](https://img.shields.io/badge/-Real--time-333333?style=flat-square) ![Payments](https://img.shields.io/badge/-Payments-333333?style=flat-square) ![Voting](https://img.shields.io/badge/-Voting-333333?style=flat-square)

<br/>

**CheckIn — Anti-Fraud Attendance System**

*Next.js · Prisma · Neon · Clerk*

A GPS-based attendance system for the Mastercard Foundation Scholars Program at KNUST, built on a five-layer anti-fraud architecture — Student ID, OTP verification, GPS geofencing, time-window locks, and device fingerprinting.

![Anti-Fraud](https://img.shields.io/badge/-Anti--Fraud-333333?style=flat-square) ![GPS](https://img.shields.io/badge/-GPS-333333?style=flat-square) ![Attendance](https://img.shields.io/badge/-Attendance-333333?style=flat-square) ![Full-Stack](https://img.shields.io/badge/-Full--Stack-333333?style=flat-square)

<br/>

**BotForge — No-Code AI Chatbot Builder**

*RAG · pgvector · Claude API · Paystack*

A no-code platform for building AI chatbots, with a retrieval-augmented generation engine, pgvector embeddings, and an embeddable widget — billed through Paystack.

![AI](https://img.shields.io/badge/-AI-333333?style=flat-square) ![RAG](https://img.shields.io/badge/-RAG-333333?style=flat-square) ![No--Code](https://img.shields.io/badge/-No--Code-333333?style=flat-square) ![Chatbots](https://img.shields.io/badge/-Chatbots-333333?style=flat-square)
